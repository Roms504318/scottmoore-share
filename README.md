# scottmoore-share

Private static-site repo for Scottmoore LLC, deployed to Vercel at **share.sco-mo.com**.

Each subfolder is a standalone HTML deliverable served at `share.sco-mo.com/<folder-name>`. The root landing page intentionally reveals nothing — access to individual deliverables requires a direct link.

## Folder-per-deliverable convention

| Folder | URL | Description |
|---|---|---|
| `hgi-rail-punchlist/` | `share.sco-mo.com/hgi-rail-punchlist` | HGI rail & bench punch list tracker |

## Workflow

Add a new folder with index.html, commit, push — Vercel auto-deploys.
