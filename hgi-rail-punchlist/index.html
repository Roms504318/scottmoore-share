Punch List Tracker

Shareable punch list with initials and timestamp tracking for both completion and approval. This version saves data in the browser on the device being used.
Important: if you host this as a simple HTML page on your domain, each device keeps its own saved version. If you want one live shared list where everyone sees the same updates, this same layout needs a small database behind it.
